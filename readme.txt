== Description ==

Always get the latest Bundesliga Table using this plugin.

This plugin connects to www.bundesligatable.com to get the latest Bundesliga table. This service is offered for free.

**What's new**

* The plugin is brand new.

== Installation ==

1. You need to load/activate the curl extension. If you do't know what this means, just go on. Most servers have curl installed!
2. Download the plugin and extract its contents.
3. Upload the `bundesliga-table` folder to the `/wp-content/plugins/` directory.
4. Activate `Bundesliga Table` plugin through the 'Plugins' menu in WordPress.
5. In your admin console, go to Appeareance > Widgets, drag the `Bundesliga Table` widget to wherever you want it to

That's it!
